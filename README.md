# Agentic AI: Building Autonomous Workflows

This repository contains notes, code implementations, and key patterns from the [DeepLearning.AI Agentic AI course](https://learn.deeplearning.ai) taught by **Andrew Ng**. 

The course focuses on shifting from simple "prompt-response" interactions to building sophisticated systems where LLMs act as the central "reasoning engine" to manage complex, multi-step workflows.

## 🚀 Course Overview
Agentic AI moves beyond static LLM outputs by giving models the autonomy to plan, use tools, and iterate on their own work. This course provides a vendor-neutral, first-principles approach to building these systems using **Python** and advanced design patterns.

## 🧠 Key Agentic Design Patterns
The curriculum is structured around four primary patterns that enable AI to handle open-ended tasks:

*   **Reflection:** Implementing feedback loops where an agent critiques its own plan or output to iterate and improve (e.g., a "Self-Correcting Coder").
*   **Tool Use:** Enabling agents to interact with the real world by calling APIs, performing web searches, or executing code.
*   **Planning:** Moving beyond hardcoded workflows to allow agents to dynamically determine the sequence of actions needed to achieve a goal.
*   **Multi-Agent Collaboration:** Coordinating multiple specialized agents (e.g., a Manager, a Writer, and a Reviewer) to solve problems through hierarchical or peer-to-peer communication.

## 🛠️ Skills & Concepts Covered
*   **Task Decomposition:** Breaking down complex business goals into modular, agentic steps.
*   **Systematic Evals:** Moving from "vibe-based" prompting to data-driven error analysis and performance metrics.
*   **Autonomous Research:** Building end-to-end agents capable of searching, synthesizing, and reporting on complex topics.
*   **Human-in-the-Loop:** Integrating human oversight into autonomous workflows to ensure reliability and safety.

## 📖 Repository Structure
- `/notebooks`: Implementation of Reflection, Tool Use, and Planning patterns.
- `/scripts`: Python scripts for autonomous research agents.
- `/evals`: Frameworks for testing agentic performance and accuracy.
